# Bookbot
:book:

## :toolbox: Requirements:  
This project was done with python-3.8.10 and aiogram-2.24  

## :link: Download & Run
Clone repository:
```
git clone https://github.com/shuygena/Bookbot bookbot
```
Go to directory:
```
cd bookbot
```
Install requirements:  
```
python3 -m pip install -r requirements.txt
```
Run bot:   
```
python3 bot.py
```
